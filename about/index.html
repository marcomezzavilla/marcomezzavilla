Hi, I'm Marco. 26 y/o. Italy. Designer. Creative enthusiast. Proudly Work in Progress…

What I Do

I develop handcrafted websites, pixel by pixel.

I use HTML5 and CSS3. When necessary, I will use secure and light CMS, in order to entrust the management and updating of the site directly to the customer.

I choose time by time the most suitable tool for the purpose, avoiding any kind of "prefab" material.

I prefer to handle all of the graphics, in order to create unified and consistent experiences.

What I Do Not Do

I don't work with software that I know superficially, because in case of last-minute changes or special requests I wouldn't be able to ensure an acceptable quality level.

I'm not a programmer: when you need to, I can implement code — mostly javascript — written by professionals.

I don't and never will use Flash.

I don't regardlessly accept all work requests, and I will definitely cut off proposals with no expected compensation.
